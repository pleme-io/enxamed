# enxamed

[![crates.io](https://img.shields.io/crates/v/enxamed.svg)](https://crates.io/crates/enxamed)
[![license](https://img.shields.io/badge/license-MIT-blue.svg)](./LICENSE)

enxamed — the ENXAME BitTorrent daemon/client: an async (tokio) shell that wires the sans-io enxame-engine to tracker announce, TCP peer connections, and disk. Part of the pleme-io ENXAME suite.

Part of **ENXAME** — the pleme-io BitTorrent suite: one sans-io engine and
thin faces, with GUIs on every platform (terminal, web, native desktop).
See [`theory/ENXAME.md`](https://github.com/pleme-io/theory/blob/main/ENXAME.md).

## Use

```toml
enxamed = "0.1"
```
